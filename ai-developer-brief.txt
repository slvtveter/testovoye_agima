Конкурсное задание: AI-ассистент для команды

Задача
За 2 часа создать веб-сервис — мини-дашборд, который помогает команде быстро получать информацию через AI.

Что нужно сделать

Функционал
- Поле ввода для вопроса
- Кнопка "Спросить AI"
- Ответ отображается в красивом виде
- История последних 5 запросов
- Кнопка "Копировать ответ"

Технологии на выбор
- React / Next.js / Vue / Svelte
- Node.js / Python или чисто клиентский вариант с API
- Любые AI API: OpenAI, Claude и другие

Что проверяем
- умение структурировать проект и писать чистый код
- навыки вайб-кодинга: скорость и качество генерации через AI
- UI/UX чувство
- презентацию решения

Видео
Вам необходимо записать 3-5 минутное видео и показать:
- как работает сервис
- какие решения вы приняли и почему

Оценивание
- Работает ли сервис — 25
- Качество UI/UX — 25
- Чистота кода — 20
- Презентация и объяснение решений — 20
- Креативность — 10
- Подключение реальной LLM с настроенными промптами — +10

Дедлайн
- 2 часа на код
- 24 часа на видео

Что не считается правильной LLM
- Просто вызов API без промпта
- Ответы как ChatGPT без специализации
- Нет обработки ошибок — просто crashed
